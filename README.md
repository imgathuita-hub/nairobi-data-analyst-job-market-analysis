# Nairobi Data Analyst Job Market Analysis 2026

**Understanding the current demand for Data Analysts in Nairobi**

## 📋 Project Objective
This project analyzes **32 recent Data Analyst, BI Analyst, and related job postings** in Nairobi (collected in May 2026) to uncover skills demand, hiring trends, and actionable insights for job seekers.

## 🛠️ Tools & Technologies
- **Python** (Pandas) – Data scraping, cleaning, and feature engineering
- **Power BI** – Interactive dashboard creation
- Excel – Initial data collection

## 📊 Key Insights

- **SQL** is the most demanded skill, appearing in **53%** of jobs
- **Power BI** (13 jobs) and **Python** (16 jobs) are strong differentiators
- The average job requires **3.2 key skills**
- **Fintech** and **NGOs** are the most active sectors hiring data professionals
- Most roles fall under Mid-level, but several Junior positions are available
- Salary transparency remains low — only a few postings disclosed salary ranges

## 🎯 Recommendations for Job Seekers in Nairobi

1. **Master the Core Trio**: SQL + Excel + Power BI (highest frequency combination)
2. Learn **Python** to significantly stand out
3. Target **Fintech** companies for better opportunities
4. Build practical projects using local Kenyan data
5. Create a strong portfolio showcasing both technical skills and business understanding
